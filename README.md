# Timer Task

Module for Angular2

Here is a [live demo](http://timertask.yoannck.com/) of the module.

#Example

```javascript
import { TimerTaskModule } from './timer-task/timer-task.module';

@NgModule({
  ...
  imports: [
    TimerTaskModule
  ],
  ...
})
```

## Install

Run `npm install git://github.com/yoannck/timer-task.git`
